# Pomoduc

A minimalist, professional-grade Pomodoro timer designed for deep focus and detailed productivity tracking.

## üöÄ Core Features

### 1. Dual View Modes
- **Full View**: A comprehensive workspace with a timer, task manager, and professional statistics.
- **Mini Mode**: A sleek, compact horizontal bar that stays out of your way. Includes **custom transparency controls** for a true "stay-on-top" experience.

### 2. Professional Stats Interface
- **Calendar Visualization**: Precise tracking of focus sessions via daily and weekly views.
- **Interactive Timeline**: Use **Ctrl + Scroll** to freely zoom in/out on the timeline for micro-details or high-level overviews.
- **Weekly Bar Summary**: A fixed top row showing daily focus totals at a glance.
- **Custom Chronology**: Starts the week on **Monday** and reflects your personal workday hours.

### 3. Smart Customization
- **Personalized Workday**: Set your day start/end times; statistics will automatically align with your schedule.
- **Task Management**: Create and track specific tasks with real-time duration summaries.
- **Aesthetic Design**: Modern glassmorphism UI with subtle animations and a clean typography system.
